Run Guide
=========

Needed files in one folder:
- Main.java
- Library.java
- Book.java
- Member.java
- IssueRecord.java

Windows / Linux
---------------
1) Open terminal in this folder
2) Check Java tools
   java -version
   javac -version
3) Compile code
   javac *.java
4) Start app
   java Main

If `javac` is missing on Ubuntu:
   sudo apt update
   sudo apt install default-jdk

Quick note:
- Keep all `.java` files together.
- If you add `package`, update run commands too.
